Korok Engine is a free lightweight, cross-platform, component-based 2D game engine written in Golang and released under the zlib license. Greatly inspired by bitsquid blog

Alpha release features:

  • Entity management
  • Font/Mesh rendering
  • Batch sprite rendering
  • Keyboard input process
  • FPS/Debug draw

There is no document for alpha release, but there is a demo-project, which show all the feature listed above.

Github

Note:
Alpha release is not feature complete and with bugs, it mainly to verify the idea behind the engine, and it works well.
The next version is beta:

  • Tag/Script system
  • Transform based node system
  • Flexible dimension
  • Layer/Z-order support
  • Particle system
  • and less bugs
