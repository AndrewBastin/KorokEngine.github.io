Korok Engine is a free lightweight, cross-platform, component-based 2D game engine written in Golang and released under the zlib license. Greatly inspired by bitsquid blog

Beta release features:

  • Tag/Script system
  • Transform based node system
  • Z-Order support
  • Particle system
  • Sprite/Tween animation system
  • GUI system

There is no document for beta release, but there is a demo-project, which show all the feature listed above.
